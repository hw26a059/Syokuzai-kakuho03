# 食材目利きゲーム 仕様書：全体概要

## 1. ゲーム概要
本作は、お肉、お魚、野菜の「目利き（鮮度や旬の選び方）」をテーマにした、クイズ形式のカジュアルゲームです。
日常の買い物に役立つ実用的な知識を手軽に楽しく学ぶことができます。

## 2. ターゲット層
- 主婦・主夫
- 一人暮らしを始めたばかりの人
- 食材の選び方や旬に興味がある一般消費者

## 3. 主要コンセプト
- **「目利き」を楽しく学ぶ**: クイズを通じて、食材の鮮度や美味しい「旬」の見分け方を習得する。
- **学びが料理に繋がる**: 獲得した食材を使って料理を行い、クイズの成績がそのまま料理の出来栄えに直結する。
- **コレクションと競争**: 図鑑を埋める達成感と、ハイスコアによるランキング競争で継続的なモチベーションを提供する。

## 4. 全体ゲームフロー
1. **食材の選択・クイズ（目利き）**：お肉、お魚、野菜から食材を選び、鮮度や旬に関するクイズに挑戦する。
2. **クイズの判定・ミニ解説**：正誤判定とともに、食材選びのコツを学ぶミニ解説を閲覧。
3. **料理フェーズ**：手に入れた食材を用いて料理を調理。クイズの正誤によって料理の美味しさ（評価）が変化。
4. **図鑑登録・スコアリング**：完成した料理は「料理図鑑」に登録される。獲得スコアはランキングへ送信。
